Description:

	Current learning project to introduce myself to UE4 programming.
A third person game where the character is constantly losing 'charge',
which must be refilled by collecting batteries (Key: E). As charge is lost
the character will slow down and dim. As charge is gained the character 
will speed up and increase in lumosity.
